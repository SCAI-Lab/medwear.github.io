MEDWEAR Physiological Signal Schemas

Below are the core MEDWEAR message types representing raw sensor data for common physiological signals. Each links to downloadable JSON Schema and ROS 2 message definitions.

  • Ballistocardiogram (BCG) — JSON Schema | ROS 2 .msg
  • Electrocardiogram (ECG) — JSON Schema | ROS 2 .msg
  • Electrodermal Activity (EDA) — JSON Schema | ROS 2 .msg
  • Electroencephalogram (EEG) — JSON Schema | ROS 2 .msg
  • Electromyography (EMG) — JSON Schema | ROS 2 .msg
  • Electrooculogram (EOG) — JSON Schema | ROS 2 .msg
  • Impedance Cardiogram (ICG) — JSON Schema | ROS 2 .msg
  • Photoplethysmogram (PPG) — JSON Schema | ROS 2 .msg

All schemas and message definitions are maintained in our GitHub repository.
