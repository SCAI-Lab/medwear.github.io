Standardization Framework

MEDWEAR builds upon existing efforts like Open mHealth and extends them for raw sensor data. We provide:

  • Data schemas for physiological signals (e.g., ECG, EMG, PPG)
  • ROS 2 message definitions for robotics and real-time systems
  • Conversion tools for interoperability

Design Principles

  • Vendor-agnostic and modular
  • Machine-learning ready formats
  • Support for clinical workflows and real-time use
